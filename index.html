Welcome to my portfolio.

I am an aspiring Junior Developer from the Cleveland/Akron, OH area. I'm currently enrolled in the We Can Code IT Full-stack developer program in Cleveland.

Ever since I was a kid, I've always been fascinated with puzzles. I liked making my brain work to find solutions to everything within reach, and I loved taking things apart and putting them back together to see how (and if) they worked.

I eventually learned to look at each new challenge in life not as merely a problem, but as a puzzle to solve -- which led me to where I am today, as an aspiring junior developer. I carry that love for problem solving and attention to detail into my code and am excited to learn more as I continue my journey.

My Projects

Here is a small sample of a couple projects I've accomplished. A full list of projects completed on GitHub can be found here, and each picture will take you to the respective project on GitHub.

Wardrobe Organizer web app

This was a simple web app built using HTML, CSS, and Bootstrap on the front-end, and utilizing MVC and MS SQL Server on the back-end. I loved this project because it allowed for more freedom than other projects in the past, and allowed me to showcase some of my personality.

I tried to go for a more refined look with the styling, and plan on adding more functionality in the future based around the database integration features.

Tuscany Villa Static Web Page

This was a showcase of ability to copy a design concept and implement it as a static web page. It is designed as a landing page, so database functionality isn't built in. It's fully fleshed-out with HTML, CSS, and Bootstrap, allowing it to me migrated to MVC for full web app functionality if desired.

Bank Account Console Application

This was a console application built in C# utilizing the concepts of OOP (Object-Oriented Programming), inheritance, and polymorphism. It is designed to take user information and handle simple transactions, but provides a good base for a more in- depth web app that could be improved with front-end functionality.

It was a very "meaty" project, with a lot of back-end work -- exactly the kinds of situations I thrive in.
